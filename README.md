## 📖 About The Project

This repository contains the source code for the **Jarvis Official Website**, built using pure HTML, CSS, and JavaScript. 

Originally, this website was designed for clients to purchase the Jarvis AI software and request custom AI workflow integrations. While the commercial operations are currently on pause, this codebase has been made public as a **free-to-use template**. Any developer looking for a sleek tech/AI product website template can easily adapt this for their own projects!

### ✨ Features
* 🖥️ **Modern UI:** Clean, futuristic design tailored for AI and tech products.
* 🔐 **Admin Dashboard:** Built-in admin login portal.
* 📥 **Legacy Downloads:** Portal to download older versions of the Jarvis software directly from the site.
* 🛠️ **Custom AI Workflows:** Showcases services for building bespoke AI solutions.
